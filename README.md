### Base URL
```
http://localhost:5000
```

### Endpoints

#### `POST /api/classify`
Classify emergency from natural language description.

**Request:**
```json
{
  "userDescription": "Fire in building"
}
```

**Response:**
```json
{
  "emergencyType": "fire",
  "severity": "CRITICAL",
  "explanation": "Flames or fire in a building poses an immediate threat to life and property.",
  "firstAidSteps": [
    "Evacuate the building immediately",
    "Call emergency services",
    "Do not use elevators",
    "Stay low to avoid smoke inhalation"
  ],
  "nearbyResources": [
    {
      "type": "Fire Rescue",
      "name": "Rapid Response Fire Team",
      "distance": 1.2,
      "eta": 3,
      "status": "Dispatched",
      "lat": 13.0750,
      "lng": 80.2650
    },
    {
      "type": "Fire Station",
      "name": "Central Fire Station",
      "distance": 1.8,
      "eta": 5,
      "lat": 13.0878,
      "lng": 80.2785
    }
  ]
}
```

#### `GET /api/health`
Health check endpoint.

**Response:**
```json
{
  "status": "ok",
  "timestamp": "2026-01-21T12:00:00.000Z",
  "storage": "in-memory",
  "totalReports": 0,
  "aiEnabled": true
}
```

#### `POST /api/report`
Submit emergency report (stores in memory).

#### `GET /api/reports`
Retrieve all stored reports.

#### `GET /api/stats`
Get system statistics.
